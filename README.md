# Pay Center API

支付中心 API 接口定义和文档仓库。

## 功能

- Protocol Buffers 接口定义
- Swagger/OpenAPI 文档
- 数据模型定义
- API 版本管理

## 目录结构

```
pay-center-api/
├── proto/              # Protocol Buffers 定义
├── swagger/            # Swagger/OpenAPI 文档
├── models/             # 数据模型定义
└── go.mod              # Go 模块定义
```

## 使用说明

1. 定义 API 接口的 Protocol Buffers 文件
2. 生成对应的 Go 代码
3. 维护 Swagger 文档
4. 为其他仓库提供 API 定义
